# E-Buy
    An E-commerce site, developed in ASP.NET
    
## Main Features
* `Admin level`
    * `Admin login`
    * `Add new product`
    * `Add new category`
    * `Customer orders`
    * `Control business`
    * `Admin profile`

* `User level`
    * `Login`
    * `Registration`
    * `Home Page`
    * `Product`
    * `Cart`
    * `Order`
    * `Checkout`
    * `Profile`
    * `About`
    * `Contact`

## Other Functionalities
`Input validation`
* `Both client and server side validation`
<br/>

`Authentication`
* `User authentication`
* `admin authentication`
<br/>

`Authorization`
* `Control user accessiablity`
<br/>

`Cryptography`
* `Password hashing and salting`
<br/>

`Exception management`
* `Reset password throgh gmail SMTP service`
