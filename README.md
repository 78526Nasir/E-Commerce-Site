# EBuy
    An E-commerce site, developed in ASP.NET
 
 
* **Watch Ebuy video [documentation](https://www.youtube.com/watch?v=BYWlrxLEG5g).**
    
## Security Related Functionalities
* Input validation
    * Both client and server side validation
* Authentication
    * User authentication
    * admin authentication  
* Authorization
    * Control user accessiablity
* Cryptography
    * Password hashing and salting
* Exception Handling
    * All page and application level Exceptions are handled
* Password Reset Management
    * Reset password throgh gmail SMTP service
    
## Main Features
* Admin level
    * Admin login
    * Add new product
    * Add new category
    * Customer orders
    * Control business
    * Admin profile

* User level
    * Login
    * Registration
    * Home Page
    * Product
    * Cart
    * Order
    * Checkout
    * Profile
    * About
    * Contact
